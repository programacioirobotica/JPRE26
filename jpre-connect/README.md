# JPRE Connect

Base independiente para una futura web app con login, panel de usuario y canal de comunicacion con organizacion.

Objetivos de esta carpeta:
- no tocar ni depender del formulario actual
- priorizar experiencia movil
- servir como punto de partida para una app autenticada

Estado actual:
- interfaz mobile-first
- login de demostracion
- dashboard por rol
- vista de taller
- bandeja de mensajes con organizacion

Siguientes pasos recomendados:
1. conectar auth real
2. definir base de datos y roles
3. enlazar con Sheets o migrar a backend dedicado
4. convertir en PWA si encaja
